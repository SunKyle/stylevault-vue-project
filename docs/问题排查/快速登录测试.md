## 故障排除

### 如果无法跳转
1. **检查控制台错误**（F12 → Console）
2. **检查网络请求**（F12 → Network）
3. **确认用户已登录**（见上方验证步骤）
4. **检查路由**（直接在地址栏输入 http://localhost:8080/upload）

### 常见错误
- **401 Unauthorized**: 用户未登录
- **404 Not Found**: 路由配置问题
- **空白页面**: 组件加载失败
